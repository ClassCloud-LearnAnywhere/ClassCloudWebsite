This is a static website built with HTML, CSS, and vanilla JavaScript. It's optimized for GitHub Pages deployment.

### Local Development

1. Clone this repository:
   ```bash
   git clone https://github.com/ClassCloud-LearnAnywhere/ClassCloudWebsite.git
   cd ClassCloudWebsite
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (with http-server)
   npx http-server
   ```

3. Visit `http://localhost:8000` in your browser

### GitHub Pages Deployment

This site is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.
